 - **Option Three**:  `grep -c` 
 
> Description   **--count**
               Give the number of matching lines in a file <br> 
              [source](https://linuxcommand.org/lc3_man_pages/grep1.html)   
          
              
 **Example 1**  
 ```

```
**Explaination** : The command counts the number of lines in *HandRHawaii.txt* that contains the word "Kalia".
 
 **Example 2**
